# Pravas
Pravas is a lightweight PHP-based Bus Ticket Booking System designed to simplify bus reservation and travel management. 
The application provides an easy-to-use interface for passengers to search, book, and manage tickets, while administrators can efficiently manage buses, routes, schedules, and bookings.

## Features

### For Users:
* User registration and login
* Search buses by source and destination
* View available bus schedules
* Book bus tickets online
* View booking history
* Cancel booked tickets
* Download and view tickets

### For Admins:
* Admin authentication and dashboard
* Manage buses and routes
* Manage schedules 
* View and manage bookings
* Manage registered users
* Generate booking and revenue reports
* Monitor overall system activity

## Technologies Used
* PHP 8+
* MySQL
* HTML5
* CSS3
* Bootstrap 5
* JavaScript
